회원가입 절차

내방신청 시스템을 이용하기 위해서는 회원가입 절차를 거쳐야 합니다.
회원가입시 휴대폰 인증을 하셔야하며 휴대폰 인증이 불가능한 경우에는 임직원 인증을 거친 후 회원 가입이 가능합니다.
개인정보, 소속 회사 정보등이 변경되는 경우 "회원정보수정" 메뉴에서 수정가능합니다.

1. 내방신청 홈페이지 접속

- PC용 ( https://visit.samsungshi.com )
- 모바일용 ( https://mvisit.samsungshi.com )

2. 회원가입 메뉴 접속

보안교육은 온라인을 통해 바로 받을 수 있으며
반드시 한 화면에서 5초 이상 학습 후 이동 가능합니다.

3. 이용약관 동의

내방 신청 시 소속회사명을 변경등록 할 수 있습니다.
방문처 담당자가 신청해주는 대신 신청정보를 승인하므로
반드시 방문처 기입 바랍니다.

4. 휴대폰 실명 인증

임직원 승인시 내방신청이 완료되며,
핸드폰과 이메일로 안내받으실 수 있습니다.

5. 회원가입 정보 입력

내방신청한 해당 장소로 방문하시면 됩니다.
